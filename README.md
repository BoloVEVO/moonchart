# Moonchart

## What is Moonchart?
Moonchart is a Haxelib backend tool designed to manage the chart file formats of various rhythm games at the same time.<br>
Can be used to detect, load, and convert between multiple formats efficiently.

## Available formats
| Format               |
|----------------------|
| FNF (Legacy)         |
| FNF (Psych Engine)   |
| FNF (FPS +)          |
| FNF (Ludum Dare)     |
| FNF (V-Slice)        |
| Guitar Hero          |
| Osu! Mania           |
| Quaver               |
| StepMania            |
